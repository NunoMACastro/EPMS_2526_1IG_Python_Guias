# Material de Apoio - Informática de Gestão

Materiais de apoio para aulas de Informática de Gestão (10.º ano). O repositório
está dividido em duas áreas principais: Python e Git.

## Conteúdo

- [Python](Python/README.md) — apontamentos e exercícios graduais de Python
  (10 módulos, do básico a algoritmos).
- [Memória](Memoria/README.md) — teoria guiada sobre memória do computador,
  gestão de memória em Python e estruturas dinâmicas.
- [Guia de Git](Git/git-guia.md) — comandos essenciais, conceitos e fluxos
  recomendados para alunos.
- [Template .gitignore](Git/template-gitignore.md) — base pronta para vários
  stacks, com notas de boas práticas.
